Tablentor is a table builder and designer addon of Elementor.

== Description ==
Tablentor is a table builder addon of elemenotr. You can build and design your table as you want.

== Installation ==

1. Install Tablentor using the native Plugin installer, or download the zip and extract it in the wp-content/plugins/ directory.
2. Activate the plugin through the ‘Plugins’ 3. 
3. You can now find the widgets in the Elementor editor basic category.
