== Installation ==

= Automatic Installation =
1. Log in to your WordPress admin dashboard
2. Navigate to **Plugins > Add New**
3. Search for "Tablentor"
4. Click **Install Now** and then **Activate**
5. The Tablentor widgets will appear in your Elementor editor under the **Basic** category

= Manual Installation =
1. Download the Tablentor plugin zip file
2. Log in to your WordPress admin dashboard
3. Navigate to **Plugins > Add New > Upload Plugin**
4. Choose the downloaded zip file and click **Install Now**
5. Activate the plugin through the **Plugins** menu
6. Start using Tablentor widgets in your Elementor editor

= After Installation =
1. Edit any page/post with Elementor
2. Search for "Basic Table" or "Table CSV" in the widget panel
3. Drag and drop the widget to your desired location
4. Configure your table settings and content
5. Publish your page to see your beautiful table in action

== Frequently Asked Questions ==

= Does Tablentor require Elementor? =

Yes, Tablentor is specifically designed as an Elementor addon and requires Elementor (free version) to be installed and activated.

= Does it work with Elementor Pro? =

Absolutely! Tablentor works seamlessly with both free Elementor and Elementor Pro versions.

= Do I need any other plugins? =

No additional plugins are required. Tablentor works independently with just Elementor installed.

= How many columns can I create? =

You can create up to 20 columns in your tables. This limit ensures optimal performance while accommodating most use cases.

= Is there a limit on the number of rows? =

No, you can add unlimited rows to your tables. The plugin is optimized to handle large datasets efficiently.

= Can I import data from Excel files? =

Currently, Tablentor supports CSV file imports. You can easily convert Excel files to CSV format and import them into your tables.

= Are the tables responsive? =

Yes! All tables created with Tablentor are fully responsive and automatically adapt to different screen sizes and devices.

= Can I customize the table design? =

Absolutely! Tablentor provides extensive styling options including colors, typography, borders, spacing, and more to match your website's design.

= Does it support search functionality? =

Yes, you can enable live search functionality that allows visitors to search through table content in real-time.

= Is Tablentor translation ready? =

Yes, Tablentor is fully translation ready and includes proper text domains for easy localization.

= Can I use images in table cells? =

Yes, you can include images in your table cells with full control over image dimensions and styling.

= Does it work with caching plugins? =

Yes, Tablentor is compatible with popular caching plugins and Elementor's built-in caching system.
